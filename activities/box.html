block tags

Block tags create a block level element that occupies the full width availiable, Inline tags create an inline element that does not on a new line.

A margin is the space outside an element creating distance between that element and other element, a border is a line that surrounds an element, and badding is the space inside an element.

Absolute units px pt cm, reletive units em rem %, Viewport units vw,vh,vmin

The figure tag is used to show the content that is referenced from the main content.
