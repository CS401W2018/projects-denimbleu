Structual Tags

Header

This is the top of the page that never changes even when the page changes.

Nav

This is what allows the links to be on the page and to navigate the webpage. It can go in the header but also can go out of the header. The links do not go to another website but simply go to a different page.

Main

This is what the page is about.There usally is a different main for the different pages on the websights

Footer

This is where the copyright usally goes. Or a privacy notice, maybe address, phone number, other links as well. Just things that could be helpful.

HTML Validation

This checks your website to see if it has any errors or if it is legal to use. It checks to see if it follows the guidlines of the Web.
