Why do we need a viewport meta tag? It is necessary for designing web pages for mobile devices of different sizes. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? The one that wins is the tag that was last instructed. How do you change the CSS for a web page displayed on a phone? You use @media screen and (max-width set to whatever screen size) under that you are able to treat it as if it is a new css page just for a smaller screen
